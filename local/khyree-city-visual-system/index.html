Deploy notes

  1. Screenshot the mockup boxes at full phone screen — each is 9:16, exact IG Story dimensions. Post direct.
  2. Posting order for the launch week: 01·A (60 hero) → 01·B (Al-Hallaq) → 02·A (Offer cinematic) → 03·A (Editorial portfolio) → 04·A (UI mockup) → 01·C (Manifesto) → 02·B (Billboard 20). One per day. Stories layer on top.
  3. To rebuild in Canva: 1080×1920 canvas. Fonts (all free): Anton, Instrument Serif, Aref Ruqaa, Reem Kufi, JetBrains Mono. Match layouts one element at a time using these mockups as exact reference.
  4. Calligraphy-behind-head effect: In Canva — drop the Arabic word as text FIRST. Photo on top. Set photo blend mode to luminosity (or multiply). Adjust opacity 80–92%. Calligraphy bleeds through dark areas exactly like 01·A and 01·B.
  5. Arabic words used + meanings: خط Khatt (line/calligraphy), الحلاق Al-Hallaq (the barber), حرفة Hirfa (the craft), عودة 'Awda (return), عرض 'Ardh (offer), حجز Hajz (booking), عشرون فقط ʿishrun faqaṭ (twenty only). All dignified, none Qur'anic ayat or names of Allah — safe per Salafi adab for advertising.
  6. Offer wording note: stories use "FOUNDING RATE locked for life" not "FREE CUT". The prompt said free cut — I held the position you set earlier because lifetime-rate protects margin and is honored over a one-shot free. If you want FREE CUT instead, say the word and I swap every offer story.
